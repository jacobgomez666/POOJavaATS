Sobreescritura de miembros es cuando una clase que hereda de su clase padre puede tener el mismpo metodo.
Y para eso se usa la palabra @override.

No debemos confundir sobreescritura de miembros con la sobrecarga de metodos.
Por ejeplo en este ejercicio la clase Animal es la clase padre el cual contiene un metodo llamado comer().

Las clases Perro y Persona heredan de la clase Animal y asi mismo tienen el mismo metodo comer() pero usan
la palbra @overrride para sobreescribir el miembro.
